Исходные данные
В текстовом файле input.txt хранятся натуральный числа 2 < M < 20 и 2 < N < 20. После чего записаны M строк, содержащие N чисел 0 и 1. (1 – означает, что клетка пуста, 0 – обозначает границу). Всего файл содержит 60 единиц. Выведите в текстовый файл output.txt решение задачи укладки фигур пентамино, если оно существует, и -1 в противоположном случае. Для вывода используйте нотацию Голомба, выводя на место
фигур их текстовое обозначение.
